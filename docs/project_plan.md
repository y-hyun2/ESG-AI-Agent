# 📘 ESG Insight Agent – 프로젝트 기획서

## 1. 프로젝트 정의

### 🎯 프로젝트 목표
ESG Insight Agent는 건설사를 위한 **ESG 문서 분석·리스크 진단·보고서 생성 자동화 시스템**을 구축하는 것을 목표로 한다.  
K-ESG, SASB, GRI 등 공개 ESG 문서를 활용하여 **ESG 평가의 정확성·속도·효율성을 높이는 AI Agent**를 개발한다.

### 🧩 주요 기능
- ESG 문서 수집 및 전처리  
- ChromaDB 기반 Vector DB 구축  
- RAG 기반 ESG 문서 검색 및 요약  
- 정책 비교(K-ESG / SASB / GRI)  
- 프로젝트 기반 ESG 리스크 진단  
- K-ESG 61개 항목 기반 ESG 보고서 자동 생성  
- 규제 변경 자동 모니터링  
- React 기반 대시보드 UI 제공

## 6. 요구사항 정의

### 6.1 기능 요구사항
- ESG 문서 자동 전처리 기능  
- Vector DB 기반 검색 및 요약  
- 문서 비교 및 분석 기능  
- 프로젝트 기반 ESG 리스크 진단  
- K-ESG 기반 보고서 자동 생성  
- 규제 변경 감지 기능  
- UI에서 결과 확인 가능  

### 6.2 비기능 요구사항
- 정확도: RAG 검색 품질 85% 이상  
- 응답 속도: 주요 기능 10초 이내  
- 확장성: ESG 데이터 추가 가능  
- 안정성: 오류 발생 시 graceful fallback 적용
